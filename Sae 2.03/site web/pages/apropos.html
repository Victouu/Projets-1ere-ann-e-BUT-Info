À propos

Le site de la SAE 2.03 est un site web fictif réalisé dans le cadre d'un projet scolaire. Il a pour but de servir d'exposition pour le systeme de connexion de la SAE 2.03.

Les créateurs

Le site a été réalisé par Lilian BROSSARD, Baptiste FROLLA, Victor ROUE et Yann CHESNEL.

Les technologies utilisées

Le site a été réalisé en HTML, CSS et JavaScript. Le systeme de connexion utilise PHP et SQL

Les sources

Les svg et icônes les proviennent de SVGRepo.
